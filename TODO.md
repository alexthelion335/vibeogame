# Feature Requests

Here is a list of requested features to implement.

- Add shield option to buy list. Allow this item to shield the player for 10 seconds, and it can only be used once. After use, the player has to buy it again to use it again.
- Add scythe melee option to buy list. This should be able swipe across multiple chickens within a certain radius of the player.
- Add highscore screen where you can put in your initials at the end, display the highscores on the main menu
- Make grenade launcher cause more damage to the chickens, it should fully kill the chickens if they are in the blast radius. Can you also update the name of the weapon in the UI to indicate that it is a launcher, not just one grenade.
