# Feature Requests

Here is a list of requested features to implement.

- Add a 25 percent chance of a med pack being spawned after each wave. This med pack will fully regenerate the health of the player. If the game is in multiplayer mode, spawn two med packs at the same time for the same 25 percent chance after each wave.
- When chickens die, make red confetti explode in the same spot the chicken was.
- Create a currency called chicken nuggets. When a white chicken dies, make it drop 1 chicken nugget, and when a brown chicken dies, make it drop 2 chicken nuggets.
- Add a 10 second time period between waves where there is a shop screen. The screen should disappear after this period.
- The shop screen should give the option to upgrade the original weapon's rate of fire by 10 percent for 25 nuggets.
- There should also be other weapons to buy like a potato shotgun (maybe costs 100 nuggets) and a grenade that has a certain blast radius (maybe 50 nuggets)
- Implement these weapons in the game
- Add an inventory system where you can switch between your weapons using the number keys, and if you press E you can open a menu where you can rearrange where these weapons go.
